# Smart AI Expert

A free multi-model AI router that:
- Classifies intent (coding / logic / creative / quick answers)
- Selects a lead model per task type
- Calls support models in parallel
- Combines answers into one expert response

## Features
- Lead + support model strategy
- Show intent + lead model
- Show individual model answers
- Copy answer button
- Light/Dark theme UI
